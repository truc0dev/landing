Projects.

A selection of things I've been building — some are live, others are still in the making. This space includes shipped work, experimental concepts, collaborative prototypes, design-led iterations, backend integrations, and unfinished ideas that continue to evolve.

  • TrazaNet / TrazaMovil

    Web app designed for livestock traceability and administrative tasks linked to national agricultural systems. Acts as a front-end for GeoMu, enabling geolocation services and deep learning-based livestock analytics for animal behavior, health, and performance.

    In development – PWA + future native app

  • GeoMu

    Livestock geolocation system using IoT devices with WiFi Long Range and LoRa protocols. Collects and syncs remote field data to a central database. Integrates with ML services for real-time livestock localization, anomaly detection, and predictive analytics.

    In development – Backend + Embedded systems

  • Blancanieves

    Interactive LMS with gamification and analytics built for Uruguayan institutions. Enables personalized learning experiences and real-time progress tracking for educators.

    Live – Educational platform

  • fitXpert

    Connects fitness coaches with clients. Features include customizable workout plans, nutrition tracking, and progress visualization in an intuitive interface.

    In development – Personal training app

  • Proboscis

    Contributed to a computer vision solution for precision agriculture. Worked on AI model development and data analysis to detect crop patterns using satellite and drone imagery.

    OpenCV AI Competition 2023 – 3rd Place

  • Molinari & Asociados
